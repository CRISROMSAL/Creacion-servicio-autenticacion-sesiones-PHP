Archivos principales del proyecto:
login.php
- Formulario de inicio de sesión
- Validación de credenciales
- Creación de sesión
- Redirección a bienvenida

bienvenida.php
- Protección de acceso
- Información del usuario
- Fecha y hora actual
- Mensaje personalizado para cada usuario
- Cierre de sesión

permisos.php
- Mensaje de acceso denegado
- Botón para volver al login
- Protección inversa (redirige si ya está logueado)

Conceptos teoricos que lleva el proyecto:
PHP
- session_start() - Inicio de sesiones
- $_SESSION - Variables de sesión
- $_POST - Captura de datos del formulario
- header() - Redirecciones
- exit() - Detención de ejecución
- Arrays asociativos

SASS
- Variables ($color-principal)
- Mixins (@mixin transition)
- Anidamiento 
- Módulos (@use)
- Funciones de color (color.adjust())
- Importación (@import)

CSS Avanzado
- Flexbox
- Animaciones (@keyframes)
- Transiciones
- Backdrop-filter (glassmorphism)
- Transformaciones
